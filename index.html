Required Coda tables

Create a new Coda doc and add these four tables exactly as named:

Table 1 — Reservations
Columns: Owner (text), Type (text), StartDate (date), EndDate (date), Status (text), Quarter (text), Note (text)
One row per day — StartDate and EndDate will be the same date for each row.

Table 2 — FloatRequests
Columns: Name (text), Email (text), StartDate (date), EndDate (date), Status (text), Note (text)

Table 3 — QuarterState
Columns: Quarter (text), Phase (text), JohnDone (text), RichDone (text), GregDone (text)

Table 4 — Config
Columns: Key (text), Value (text)
Add one row: Key = StartDate, Value = 2026-08-01
